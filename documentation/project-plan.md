## Objective
Build a simulated enterprise Azure environment and document the deployment process.

- Create Virtual Network

## Phase 2 - Compute

- Harden Linux VM

## Phase 3 - Identity

- Create Entra ID Groups
- Configure RBAC Roles

## Phase 4 - Monitoring

- Log Analytics Workspace

## Phase 5 - Automation
- Terraform Deployment
